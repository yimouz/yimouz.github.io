# 代码高亮测试

## JavaScript 代码
```javascript
// 这是注释 - 应该是灰色
function calculateSum(a, b) {
    const result = a + b; // const 应该是红色
    console.log("结果是: " + result); // 字符串应该是绿色
    return result; // return 应该是红色
}

// 调用函数
const numbers = [1, 2, 3, 4, 5]; // 数组应该有颜色
calculateSum(10, 20);
```

## Python 代码
```python
# 这是Python注释 - 应该是灰色
def fibonacci(n):
    """计算斐波那契数列""" # 文档字符串应该是绿色
    if n <= 1: # if 应该是红色
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# 测试代码
for i in range(10): # for, range 应该有颜色
    print(f"F({i}) = {fibonacci(i)}") # print 应该有颜色
```

/* CSS注释 - 应该是灰色 */
