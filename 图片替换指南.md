## ⚡ 一键替换脚本

如果你有多张图片需要批量处理，可以：

1. **准备图片文件夹结构**:
```
my-images/
├── avatar.jpg
├── favicon.png  
├── logo.png
├── banners/
│   ├── 0.jpg
│   ├── 1.jpg
│   └── ...
└── features/
    ├── 0.jpg
    ├── 1.jpg
    └── ...
```

2. **批量复制命令** (Windows):
```cmd
copy my-images\avatar.jpg source\medias\avatar.jpg
copy my-images\favicon.png source\favicon.png
copy my-images\logo.png source\medias\logo.png
copy my-images\banners\*.jpg source\medias\banner\
copy my-images\features\*.jpg source\medias\featureimages\
```

4. **备份原图**: 替换前先备份原始图片
